Mise en place:

Vous aurez besoin d'un logiciel pour programmer tel que Sublime Text , Atom , Visual Studio Code.


Créer un dossier /bot-discord ou vous pouvez l'appeller comme vous voulez, ce sera le dossier qui contient le bot discord.

Création du bot:

Nous avons notre espace pour notre bot il nous manque plus qu'a le créer et pour cela vous devez vous rendre sur le site de Discord et aller dans le portail développeur. Créez une application.


Donnez lui le nom que vous voulez, dans le cadre du guide l'application s'appelle discord-guide.

Nous avons besoin de copier-coller le client ID mettez le dans un bloc-note quelques part sur votre bureau.


Ensuite cliquez sur l'icone Bot

Appuyez ensuite sur le bouton Add Bot (rajouter un bot) et confirmer. Maintenant vous pouvez choisir le nom que vous voulez donner a votre bot discord, nous l'appellerons discord guide mettez lui un logo si vous le souhaitez.

On aura besoin du token du bot donc vous devriez cliquez sur le bouton pour révéler le token du bot.

Ce qui nous donnez une suite de chose bizarre mais ne soyez pas perdu ! Pas besoin de comprendre ce que sa signifie il faut juste le
copier-coller quelque part.

Permissions:

Maintenant il faut inviter le bot discord sur notre serveur discord, il y a une nouvelle fonctionnalité qui nous permer de calculer les permissions directement sur la page du bot sa ressemble a cela.

Vous devez juste cliquer sur les permissions qu'il vous faut pour votre bot, pour le guide je sélectionne juste Administrateur qui résulte en un 8.

Pour rajouter votre bot a votre serveur discord vous devez éditer ce lien https://discordapp.com/oauth2/authorize?client_id=ID_DU_BOT&scope=bot&permissions=NOMBRE_DES_PERMISSIONS

Tel que https://discordapp.com/oauth2/authorize?client_id=512981872603561984&scope=bot&permissions=8 qui sera le lien pour inviter mon bot, vous devriez juste mettre votre client ID et votre nombre de permissions. Simple non ?
